# Privacy Policy

## Data Collection

This MCP server **does not collect, store, or process any personal data**.

## What This Server Contains

All data served by this MCP is derived from publicly available Swiss federal sources:

- Direktzahlungsverordnung (DZV, SR 910.13) -- published by the Swiss Federal Council
- BLW Weisungen -- published by the Federal Office for Agriculture
- Agrarbericht -- published by the Federal Office for Agriculture
- Agate portal information -- published by BLW and cantonal agricultural offices

These are official government publications available to the public without restriction.

## No Tracking

- No analytics or telemetry
- No cookies
- No user accounts or authentication
- No IP address logging by the MCP server itself
- No request content stored

## Network Requests

When accessed via Streamable HTTP (`mcp.ansvar.eu/ch-farm-subsidies/mcp`), standard HTTP connection metadata (IP address, TLS version) is processed by the infrastructure (Cloudflare, nginx) but is not stored by the MCP server application.

When accessed via stdio (npx), no network requests are made by the server.
